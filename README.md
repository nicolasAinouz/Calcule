Il faut choisir deux nombres entre 0 et 100 et le programme ajoutera les dizaines entre les nombres.  
Par exemple si l'on a 20 et 40 le programme fera :  
20 + 30 + 40.  
Si l'on entre 90 et 120 le programme fera :
